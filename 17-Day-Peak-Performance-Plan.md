# 17-Day Peak Performance Plan

> **Apr 14 → May 1** | Based on WHOOP data

## Non-Negotiables (Every Day)

- Sleep **22:30 → 06:00**
- **160g protein**
- **10,000 steps**
- Zero nicotine
- Zero alcohol
- Morning sunlight (10 min, no sunglasses, within 30 min of waking)

---

## Supplements

| Supplement | Dose | When | Why |
|---|---|---|---|
| Magnesium Glycinate | 400mg | 30–60 min before bed | Improves deep sleep, relaxes muscles, reduces night waking |
| Zinc | 30mg | With dinner | Supports testosterone production — WHOOP shows recovery needs help |
| Vitamin D3 | 4,000 IU | Morning with food | Most people are deficient — affects energy, testosterone, mood |
| Omega-3 Fish Oil | 1,000–2,000mg EPA/DHA | With any meal | Reduces inflammation, supports heart health, improves blood flow |

---

## WHOOP Targets

| Metric | Current | Target |
|---|---|---|
| RHR | High 60s–low 70s | Below 65 bpm |
| HRV | Mid 30s–low 40s | Consistently above 40ms |
| Recovery | Mixed yellow/green | 5+ green days in a row by Apr 28 |
| Deep Sleep | 1h20m–2h16m | Above 1h45m nightly |
| Sleep Debt | Up to 2 hours | Under 30 minutes |

## Phase 1: Hard Reset — Apr 14–20

### Day 1 — Mon Apr 14

**Sleep**
- [ ] Set phone alarm for 22:00 (wind-down) and 22:30 (lights out)
- [ ] No screens after 22:00 — charge phone outside bedroom

**Nutrition**
- [ ] Last meal by 19:30, nothing after
- [ ] Stop water intake at 17:00, sip only after
- [ ] No caffeine after 12:00 noon

**Exercise**
- [ ] 10,000 steps — walk after lunch and dinner

**Lifestyle**
- [ ] 10 min morning sunlight within 30 min of waking (no sunglasses)
- [ ] Zero nicotine, zero alcohol

**Supplements**
- [ ] Buy: Magnesium Glycinate 400mg, Zinc 30mg, Vitamin D3 4000IU, Omega-3 fish oil

---

## THE DAY — Thu May 1

> **Focus: connection, not performance. Be present.**

**Morning**
- [ ] Wake naturally around 06:00. Morning sunlight. Light exercise or walk.
- [ ] Breakfast: eggs, oats, fruit, coffee (before noon)

**Nutrition**
- [ ] Lunch: chicken or fish, rice, vegetables. Normal portion.
- [ ] Dinner: LIGHT meal 2–3 hours before. Salmon + rice + veg, or chicken + sweet potato. No heavy/greasy/bloating food.
- [ ] Stay hydrated during the day. Taper water by evening.
- [ ] No caffeine after noon. ZERO alcohol (or absolute max 1 glass wine with dinner).

**Lifestyle**
- [ ] Shower, groom, fresh sheets on the bed.
- [ ] Set the mood: music, dim lighting/candles, clean space.

**Mindset**
- [ ] Focus on CONNECTION not performance. Be present. Go slow. Communicate. Ask what she likes.
- [ ] If nerves hit: box breathe (4-4-4-4) for 2 minutes. Remind yourself your body is ready — you've done the work.

    const loadData = async () => {
      try {
        const result = await window.storage.get("peak-plan-tasks");
        if (result?.value) {
          setCompletedTasks(JSON.parse(result.value));
        }
      } catch (e) {}
      setLoaded(true);
    };
    loadData();
  }, []);

  useEffect(() => {
    if (!loaded) return;
    const saveData = async () => {
      try {
        await window.storage.set("peak-plan-tasks", JSON.stringify(completedTasks));
      } catch (e) {}
    };
    saveData();
  }, [completedTasks, loaded]);

  const toggleTask = (phaseIdx, dayIdx, taskIdx) => {
    const key = `${phaseIdx}-${dayIdx}-${taskIdx}`;
    setCompletedTasks((prev) => ({ ...prev, [key]: !prev[key] }));
  };

  const getProgress = () => {
    let total = 0;
    let done = 0;
    PLAN_DATA.phases.forEach((phase, pi) => {
      phase.days.forEach((day, di) => {
        day.tasks.forEach((_, ti) => {
          total++;
          if (completedTasks[`${pi}-${di}-${ti}`]) done++;
        });
      });
    });
    return { total, done, pct: total > 0 ? Math.round((done / total) * 100) : 0 };
  };

  const getDayProgress = (phaseIdx, dayIdx) => {
    const day = PLAN_DATA.phases[phaseIdx].days[dayIdx];
    let done = 0;
    day.tasks.forEach((_, ti) => {
      if (completedTasks[`${phaseIdx}-${dayIdx}-${ti}`]) done++;
    });
    return { done, total: day.tasks.length };
  };

  const progress = getProgress();
  const currentPhase = PLAN_DATA.phases[activePhase];
  const currentDay = currentPhase.days[activeDay];

  return (
    <div style={{
      fontFamily: "'JetBrains Mono', 'SF Mono', 'Fira Code', monospace",
      background: "#0d0d0d",
      color: "#e0e0e0",
      minHeight: "100vh",
      maxWidth: 480,
      margin: "0 auto",
      padding: "0",
      overflowX: "hidden",
    }}>
      <style>{`@import url('https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@300;400;500;700&display=swap'); * { box-sizing: border-box; margin: 0; padding: 0; } ::-webkit-scrollbar { width: 4px; height: 4px; } ::-webkit-scrollbar-track { background: #1a1a1a; } ::-webkit-scrollbar-thumb { background: #333; border-radius: 2px; }`}</style>

      {/* Header */}
      <div style={{
        padding: "24px 20px 16px",
        background: "linear-gradient(180deg, #1a1a1a 0%, #0d0d0d 100%)",
        borderBottom: "1px solid #222",
      }}>
        <div style={{ fontSize: 11, color: "#555", letterSpacing: 3, textTransform: "uppercase", marginBottom: 4 }}>
          17-Day Protocol
        </div>
        <div style={{ fontSize: 22, fontWeight: 700, color: "#fff", letterSpacing: -0.5 }}>
          Peak Performance Plan
        </div>
        <div style={{ fontSize: 12, color: "#666", marginTop: 4 }}>
          Apr 14 → May 1 &nbsp;|&nbsp; Based on your WHOOP data
        </div>
        <div style={{ marginTop: 16 }}>
          <div style={{ display: "flex", justifyContent: "space-between", fontSize: 11, color: "#666", marginBottom: 6 }}>
            <span>{progress.done}/{progress.total} tasks</span>
            <span>{progress.pct}%</span>
          </div>
          <div style={{ height: 4, background: "#1a1a1a", borderRadius: 2, overflow: "hidden" }}>
            <div style={{
              height: "100%",
              width: `${progress.pct}%`,
              background: `linear-gradient(90deg, ${currentPhase.color}, ${currentPhase.color}88)`,
              borderRadius: 2,
              transition: "width 0.4s ease",
            }} />
          </div>
        </div>
      </div>

      {/* Quick access buttons */}
      <div style={{ display: "flex", gap: 8, padding: "12px 20px", borderBottom: "1px solid #1a1a1a" }}>
        <button
          onClick={() => { setShowSupplements(!showSupplements); setShowTargets(false); }}
          style={{
            flex: 1, padding: "10px 8px", fontSize: 11, fontFamily: "inherit",
            background: showSupplements ? "#1a2028" : "#141414", color: showSupplements ? "#81B4D8" : "#666",
            border: `1px solid ${showSupplements ? "#2a3038" : "#222"}`, borderRadius: 8, cursor: "pointer",
          }}
        >
          {CAT_ICONS.supplement} Supplements
        </button>
        <button
          onClick={() => { setShowTargets(!showTargets); setShowSupplements(false); }}
          style={{
            flex: 1, padding: "10px 8px", fontSize: 11, fontFamily: "inherit",
            background: showTargets ? "#251a28" : "#141414", color: showTargets ? "#C77DBA" : "#666",
            border: `1px solid ${showTargets ? "#352a38" : "#222"}`, borderRadius: 8, cursor: "pointer",
          }}
        >
          {CAT_ICONS.whoop} WHOOP Targets
        </button>
      </div>

      {/* Supplements panel */}
      {showSupplements && (
        <div style={{ padding: "12px 20px", borderBottom: "1px solid #1a1a1a" }}>
          {PLAN_DATA.supplements.map((s, i) => (
            <div key={i} style={{ padding: "10px 12px", marginBottom: 8, background: "#141414", border: "1px solid #222", borderRadius: 8 }}>
              <div style={{ fontSize: 13, fontWeight: 500, color: "#81B4D8" }}>{s.name}</div>
              <div style={{ fontSize: 11, color: "#888", marginTop: 4 }}>{s.dose} · {s.timing}</div>
              <div style={{ fontSize: 11, color: "#555", marginTop: 4, fontStyle: "italic" }}>{s.why}</div>
            </div>
          ))}
        </div>
      )}

      {/* Targets panel */}
      {showTargets && (
        <div style={{ padding: "12px 20px", borderBottom: "1px solid #1a1a1a" }}>
          {PLAN_DATA.targets.map((t, i) => (
            <div key={i} style={{
              padding: "10px 12px", marginBottom: 8, background: "#141414", border: "1px solid #222", borderRadius: 8,
              display: "flex", justifyContent: "space-between", alignItems: "center",
            }}>
              <div>
                <div style={{ fontSize: 13, fontWeight: 500, color: "#C77DBA" }}>{t.metric}</div>
                <div style={{ fontSize: 11, color: "#555", marginTop: 2 }}>Now: {t.current}</div>
              </div>
              <div style={{ fontSize: 11, color: "#2A9D8F", textAlign: "right", maxWidth: 140 }}>→ {t.target}</div>
            </div>
          ))}
        </div>
      )}

      {/* Phase tabs */}
      <div style={{ display: "flex", padding: "12px 20px 0", gap: 6 }}>
        {PLAN_DATA.phases.map((phase, idx) => (
          <button
            key={idx}
            onClick={() => { setActivePhase(idx); setActiveDay(0); }}
            style={{
              flex: 1, padding: "10px 6px", fontSize: 10, fontFamily: "inherit",
              background: activePhase === idx ? `${phase.color}15` : "transparent",
              color: activePhase === idx ? phase.color : "#555",
              border: `1px solid ${activePhase === idx ? `${phase.color}40` : "#222"}`,
              borderRadius: "8px 8px 0 0", cursor: "pointer",
              borderBottom: activePhase === idx ? `2px solid ${phase.color}` : "1px solid #222",
            }}
          >
            {phase.name.split(":")[0]}
            <div style={{ fontSize: 9, marginTop: 2, opacity: 0.6 }}>{phase.dates}</div>
          </button>
        ))}
      </div>

      {/* Day selector */}
      <div style={{ display: "flex", gap: 4, padding: "12px 20px", overflowX: "auto", borderBottom: "1px solid #1a1a1a" }}>
        {currentPhase.days.map((day, idx) => {
          const dp = getDayProgress(activePhase, idx);
          const isComplete = dp.done === dp.total;
          const isActive = activeDay === idx;
          return (
            <button
              key={idx}
              onClick={() => setActiveDay(idx)}
              style={{
                minWidth: 44, padding: "8px 6px", fontSize: 10, fontFamily: "inherit",
                background: isActive ? `${currentPhase.color}20` : isComplete ? "#0a1a0a" : "#111",
                color: isActive ? currentPhase.color : isComplete ? "#2A9D8F" : "#555",
                border: `1px solid ${isActive ? `${currentPhase.color}50` : isComplete ? "#1a3a1a" : "#1a1a1a"}`,
                borderRadius: 6, cursor: "pointer", position: "relative",
              }}
            >
              <div style={{ fontWeight: 500 }}>{day.label}</div>
              {dp.done > 0 && <div style={{ fontSize: 8, marginTop: 2, opacity: 0.7 }}>{dp.done}/{dp.total}</div>}
              {isComplete && <div style={{ position: "absolute", top: -4, right: -4, fontSize: 10 }}>✅</div>}
            </button>
          );
        })}
      </div>

      {/* Day content */}
      <div style={{ padding: "16px 20px" }}>
        <div style={{ display: "flex", justifyContent: "space-between", alignItems: "center", marginBottom: 16 }}>
          <div>
            <div style={{ fontSize: 16, fontWeight: 700, color: currentDay.highlight ? "#2A9D8F" : "#fff" }}>
              {currentDay.date}
            </div>
            {currentDay.highlight && (
              <div style={{ fontSize: 11, color: "#2A9D8F", marginTop: 2, letterSpacing: 2 }}>GAME DAY</div>
            )}
          </div>
          <div style={{ fontSize: 12, color: "#444" }}>
            {getDayProgress(activePhase, activeDay).done}/{getDayProgress(activePhase, activeDay).total}
          </div>
        </div>

        {currentDay.tasks.map((task, ti) => {
          const key = `${activePhase}-${activeDay}-${ti}`;
          const done = completedTasks[key];
          const catStyle = CAT_COLORS[task.cat] || CAT_COLORS.lifestyle;
          return (
            <div
              key={ti}
              onClick={() => toggleTask(activePhase, activeDay, ti)}
              style={{
                display: "flex", gap: 12, padding: "12px", marginBottom: 8, borderRadius: 8, cursor: "pointer",
                background: done ? "#0a0a0a" : catStyle.bg,
                border: `1px solid ${done ? "#1a1a1a" : catStyle.border}`,
                opacity: done ? 0.5 : 1,
              }}
            >
              <div style={{
                width: 22, height: 22, borderRadius: 6, flexShrink: 0, marginTop: 1,
                border: `2px solid ${done ? "#2A9D8F" : catStyle.text}40`,
                background: done ? "#2A9D8F20" : "transparent",
                display: "flex", alignItems: "center", justifyContent: "center",
                fontSize: 12, color: "#2A9D8F",
              }}>
                {done ? "✓" : ""}
              </div>
              <div style={{ flex: 1 }}>
                <div style={{
                  fontSize: 9, color: catStyle.text, letterSpacing: 1.5,
                  textTransform: "uppercase", marginBottom: 4, fontWeight: 500,
                }}>
                  {CAT_ICONS[task.cat]} {task.cat}
                </div>
                <div style={{
                  fontSize: 13, lineHeight: 1.5, color: done ? "#444" : "#ccc",
                  textDecoration: done ? "line-through" : "none",
                }}>
                  {task.text}
                </div>
              </div>
            </div>
          );
        })}
      </div>

      {/* Footer */}
      <div style={{ padding: "20px", borderTop: "1px solid #1a1a1a", fontSize: 11, color: "#333", textAlign: "center", lineHeight: 1.6 }}>
        Non-negotiables every day: Sleep 22:30–06:00 · 160g protein · 10K steps · Zero nicotine · Zero alcohol · Morning sunlight
      </div>
    </div>
  );
}
